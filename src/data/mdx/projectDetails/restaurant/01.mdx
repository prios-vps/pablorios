### Necesidad del cliente

Mi cliente es un proveedor de un sistema de ventas para comercio minorista con alrededor de 20 años en funcionamiento en multitud de locales.

Tras implementar una nueva funcionalidad que permite el uso del sistema para gestionar restaurantes, mi cliente solicitó mis servicios para crear una aplicación que pudiera facilitar la toma de pedidos de los garzones usando teléfonos inteligentes o tablets, conectándose al sistema usando la red WiFi del local restaurant donde se implementaría esta tecnología.
